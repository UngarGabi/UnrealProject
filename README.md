# Level-Based Shooter Game

Level-Based Shooter Game is an action shooter developed in Unreal Engine 5. The game is structured around multiple levels, each with different gameplay objectives, such as exploring maze-like environments or eliminating enemies spawned across the map.

The project focuses on player movement, shooting mechanics, enemy encounters, level progression and gameplay logic implemented through Unreal Engine Blueprints.

## Project Status

The game is finalized as a playable student project. It includes a functional player controller, shooting mechanics, multiple level types, enemy spawning, combat-based objectives and level progression.

## Features

- Level-based gameplay structure.
- Third-person player controller.
- Player movement and camera control.
- Shooting system implemented through Blueprints.
- Projectile-based combat.
- Multiple level types, including maze-based levels and combat-focused levels.
- Enemy spawning system across the map.
- Gameplay objectives based on eliminating enemies or completing the level.
- Collision and interaction logic.
- Basic enemy encounters and combat flow.
- Level progression system.
- Unreal Engine Blueprint-based gameplay implementation.

## Gameplay Overview

The player controls a character in a level-based environment. Depending on the level, the player may need to explore a maze, navigate through the map, or fight enemies that spawn in different areas.

In combat-focused levels, enemies are spawned across the map, and the player must find and eliminate them in order to progress. The game combines movement, shooting, exploration and objective-based gameplay.

## Controls

- `W` / `A` / `S` / `D` - Move the player
- `Mouse` - Control camera direction
- `Left Mouse Button` - Shoot

## Main Systems Implemented

### Player Controller

The game includes a player controller responsible for movement, camera handling and shooting. The player can move through the level, aim using the camera direction and shoot projectiles at enemies.

### Shooting System

The shooting system allows the player to fire projectiles during gameplay. The projectiles are used to damage or eliminate enemies and are connected to the player input system.

### Enemy Spawning System

Enemies are spawned in different areas of the map, depending on the level design. In combat levels, the player must eliminate all active enemies in order to complete the objective and progress further.

### Level-Based Progression

The game is structured around multiple levels with different gameplay goals. Some levels focus on exploration and maze navigation, while others focus on combat and enemy elimination.

### Maze and Combat Levels

The project includes different types of levels:
- Maze-based levels, where the player must navigate through the environment.
- Combat-focused levels, where enemies spawn on the map and must be defeated.

### Blueprint Gameplay Logic

The main gameplay systems were implemented using Unreal Engine Blueprints, including player input, shooting behavior, enemy spawning, collision events and level flow.

## How to Run the Project

1. Clone or download this repository.
2. Open the project in Unreal Engine 5.
3. Open the main level from the project content folder.
4. Press the `Play` button in the Unreal Editor.
